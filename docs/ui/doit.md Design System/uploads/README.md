# doit.md - Figma import package

Não é um arquivo `.fig` nativo, porque o formato `.fig` do Figma é proprietário e não pode ser gerado de forma confiável fora do Figma.

Este pacote contém SVGs vetoriais prontos para importar no Figma:

- `logo-primary.svg`
- `logo-icon.svg`
- `logo-reversed.svg`
- `logo-monochrome.svg`
- `brand-board.svg`
- `brand-tokens.json`

## Como usar no Figma

1. Abra o Figma.
2. Crie um novo arquivo.
3. Arraste os arquivos `.svg` para dentro do canvas.
4. Selecione o SVG importado e use `Ungroup` se quiser editar camadas.
5. Instale/use as fontes sugeridas:
   - Inter
   - JetBrains Mono
